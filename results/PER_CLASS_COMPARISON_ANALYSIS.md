## 1. Methodology

### 1.1 Evaluation Protocol

Per-class performance metrics were extracted from the PlantVillage validation set, which comprises 17,572 images distributed across 38 disease and healthy-plant classes. Both trained architectures — MobileNetV2 and ResNet-50 — were evaluated under identical conditions: the same validation split, the same image preprocessing pipeline (224×224 resize, ImageNet mean/std normalisation), and CPU inference without test-time augmentation. The ground truth labels correspond to the ImageFolder-sorted class ordering used during training.

For each class, the standard precision, recall, and F1-score were computed from the confusion matrix according to:

```
Precision  =  TP / (TP + FP)
Recall     =  TP / (TP + FN)
F1         =  2 * (Precision * Recall) / (Precision + Recall)
```

Macro-average metrics were computed as the unweighted mean over all 38 classes, giving equal weight to each class regardless of its support count. Weighted-average metrics additionally account for class imbalance by weighting each class by its support.

### 1.2 Comparative Framework

The per-class comparison CSV (`results/per_class_comparison.csv`) was generated by subtracting ResNet-50 F1 from MobileNetV2 F1 for each class:

```
difference  =  mobilenet_f1 − resnet_f1
```

Positive values indicate MobileNetV2 superiority; negative values indicate ResNet-50 superiority. Classes were then stratified into three tiers based on the MobileNetV2 F1 score — the primary candidate architecture — using empirically motivated thresholds:

| Tier | F1 Range | Interpretation |
|---|---|---|
| Top | ≥ 0.950 | Deployment-ready performance |
| Middle | 0.850–0.949 | Acceptable; targeted improvement possible |
| Bottom | < 0.850 | Structural failure; requires intervention |

### 2.1 Summary Table

| **Overall accuracy** | **97.13%** | 94.69% | +2.44 pp |
| Macro precision | 0.9729 | 0.9493 | +0.0236 |
| Macro recall | 0.9707 | 0.9467 | +0.0240 |
| **Macro F1** | **0.9707** | 0.9463 | +0.0244 |

| Parameters | ~2.27M | ~23.5M | MobileNet 10× smaller |
| Inference (CPU, 1 img) | not measured | not measured | — |

MobileNetV2 outperforms ResNet-50 by 2.44 percentage points on overall accuracy and by 0.0244 on macro F1, despite using approximately one-tenth the number of parameters. The near-identical macro and weighted F1 scores for MobileNetV2 (0.9707 vs. 0.9711) indicate that per-class performance is consistent regardless of class size — a sign of balanced generalisation across the dataset's 38 categories.

### 2.2 Distribution of F1 Scores

The F1 distribution across the 38 classes reveals a substantially different risk profile between the two architectures:

- **MobileNetV2:** 31 classes at F1 ≥ 0.950, 6 classes in the 0.850–0.949 range, 1 class below 0.850
- **ResNet-50:** 20 classes at F1 ≥ 0.950, 17 classes in the 0.850–0.949 range, 1 class below 0.850

MobileNetV2 places 81.6% of classes in the top tier, compared to 52.6% for ResNet-50. This means that on 11 additional classes, ResNet-50 falls below the deployment-ready threshold that MobileNetV2 clears comfortably. The long tail of ResNet-50 underperformance affects primarily heterogeneous disease classes and healthy-class variants, as detailed in Section 5.

## 3. Top-Performing Classes

### 3.1 Perfect-F1 Classes (MobileNetV2, F1 = 1.000)

Two classes achieve F1 = 1.000 under MobileNetV2, indicating zero prediction errors in both directions:

**Grape___Leaf_blight_(Isariopsis_Leaf_Spot)** (support = 430):
Both precision and recall reach 1.0000 — no false positives and no false negatives across 430 validation images. This class exhibits a visually distinctive lesion pattern: circular, darkly pigmented spots with a yellowish halo concentrated on older leaves, a phenotype that is both consistent within the class and dissimilar from all other grape diseases in the dataset. The absence of a visually similar competing class in the 38-class taxonomy appears to be the primary driver of perfect separation.

**Grape___healthy** (support = 423):
Precision and recall are both 1.0000 across 423 images. Healthy grape leaves present a consistent deep-green, lobed morphology without any disease markers. As shown in the Grad-CAM sanity analysis (Section 2 of `results/gradcam/ANALYSIS.md`), MobileNetV2 attends specifically to leaf shape and surface texture for this class, suppressing background activation entirely. The uniqueness of *Vitis* leaf morphology relative to other crop species in the dataset — its characteristic multi-lobed outline — provides a strong separating feature.

## 4. MobileNetV2 Failure Modes

### 4.1 Worst Class: Corn_(maize)___Cercospora_leaf_spot (F1 = 0.844)

This class constitutes the single most problematic category for MobileNetV2 and represents the only class below the 0.850 threshold. The per-class report reveals a strongly asymmetric error pattern:

- **Precision: 0.9838** — when MobileNetV2 predicts Cercospora, it is correct 98.4% of the time
- **Recall: 0.7390** — only 73.9% of actual Cercospora images are correctly classified; 26.1% are missed

This asymmetry indicates that MobileNetV2 has learned a highly discriminative representation for Cercospora lesions, but that representation only matches a subset of the class's visual range. The 26.1% false-negative rate corresponds to approximately **107 misclassified images** out of 410 validation samples.

The primary confusion target, as identified from the confusion matrix, is `Corn_(maize)___Northern_Leaf_Blight`, which accounts for the majority of false negatives. Both diseases produce elongated, grayish-green necrotic lesions that run parallel to corn leaf veins — Cercospora lesions are typically shorter (1–5 cm, rectangular with tan centres), while NLB lesions are longer (5–15 cm, cigar-shaped with gray-green margins). At the 224×224 resolution at which this model operates, the length differential is difficult to resolve; both manifest as parallel streaks of similar colouration. As discussed in detail in the Grad-CAM confusion analysis, the high prediction confidence in these misclassifications (0.86) indicates that MobileNetV2 is not attending to spurious features — the features it uses are genuine; they are simply insufficient to distinguish these morphologically proximate diseases.

### 4.2 Secondary Failure Classes

| Class | F1 | Primary error direction |
|---|---|---|
| Tomato___Target_Spot | 0.9034 | Confused with Tomato Early Blight (irregular necrotic spots) |
| Tomato___Early_blight | 0.9093 | Confused with Tomato Bacterial Spot and Target Spot |
| Tomato___Late_blight | 0.9166 | Cross-host confusion with Potato Late Blight |
| Tomato___Spider_mites | 0.9229 | Subtle texture signal; low-contrast mite webbing |
| Corn_(maize)___Northern_Leaf_Blight | 0.8935 | Benefits from Cercospora false positives (recall=0.994) |

A structural pattern is evident: MobileNetV2's failure mode is concentrated within the **Tomato genus** (four of the five worst-performing non-trivial classes are Tomato diseases) and the **Corn elongated-lesion** cluster. The Tomato failures share a common cause: *Alternaria* (Early Blight), *Xanthomonas* (Bacterial Spot), *Phytophthora* (Late Blight), *Alternaria tomatophila* (Target Spot), and *Tetranychus urticae* (Spider Mites) all produce spotted or streaked necrotic lesions on tomato leaves with broadly similar chromatic characteristics. The 38-class taxonomy includes six distinct tomato disease categories; at the feature level encoded by MobileNetV2's classifier head, these six categories are not always linearly separable.

## 5. ResNet-50 Failure Modes

### 5.1 Worst Class: Tomato___Late_blight (F1 = 0.796)

ResNet-50's lowest-performing class, `Tomato___Late_blight` at F1 = 0.796, is substantially worse than MobileNetV2's worst class (0.844). The error pattern is again asymmetric:

- **Precision: 0.9690** — predictions of Late Blight are correct 96.9% of the time
- **Recall: 0.6760** — only 67.6% of actual Late Blight images are identified correctly

A recall of 0.676 means ResNet-50 misclassifies **150 out of 463** Late Blight validation images — more than one in three specimens goes undetected. The high precision alongside low recall indicates that ResNet-50 has learned a narrow, highly specific feature profile for Late Blight that matches only the most prototypical examples in the class, while failing on specimens that present partial, early-stage, or variably lit lesions. The cross-host confusion with `Potato___Late_blight` (same pathogen, *Phytophthora infestans*, similar lesion morphology) accounts for a substantial portion of these false negatives, as confirmed by the confusion matrix.

Notably, the F1 gap between architectures on this class is the largest in the dataset: MobileNetV2 achieves F1 = 0.9166 versus ResNet-50's 0.7964, a **difference of +0.1202** in favour of MobileNetV2. The Grad-CAM comparison (Section 4.1 of `results/gradcam/ANALYSIS.md`) provides visual evidence for this gap: MobileNetV2 focuses its activation on the specific necrotic lesion patch and predicts correctly at 98.8% confidence, while ResNet-50 produces a diffuse, spatially incoherent activation map and misclassifies the same image.

### 5.2 Secondary Failure Classes

| Class | F1 | Primary issue |
|---|---|---|
| Tomato___Target_Spot | 0.8850 | Shared failure with MobileNet; genus-level confusion |
| Tomato___Early_blight | 0.8896 | Precision-recall imbalance (P=0.926, R=0.856) |
| Tomato___Bacterial_spot | 0.9004 | Low recall (0.840): 68 misclassified out of 425 |
| Grape___Black_rot | 0.9056 | Low recall (0.843): 74 misclassified out of 472 |
| Orange___Haunglongbing | 0.9154 | Low precision (0.844): 93 false positives |

ResNet-50's secondary failures are distributed more broadly than MobileNetV2's, affecting Grape and Orange classes in addition to the Tomato cluster. `Grape___Black_rot` is a notable example: ResNet-50 achieves recall of only 0.843 (74 false negatives out of 472), whereas MobileNetV2 achieves recall of 0.981 on the same class. The Black Rot lesion is characterised by small circular brown scorch marks with a darker border — a compact, high-contrast pattern that appears to be encoded efficiently by MobileNetV2's depthwise-separable convolutions but less reliably by ResNet-50's standard residual connections.

## 6. Class-Level Architectural Divergence

### 6.1 Classes Where MobileNetV2 Substantially Outperforms ResNet-50

MobileNetV2 leads by more than 0.05 F1 points on seven classes:

| Class | MobileNet F1 | ResNet F1 | Δ |
|---|---|---|---|
| Tomato___Late_blight | 0.9166 | 0.7964 | **+0.1202** |
| Grape___Black_rot | 0.9883 | 0.9056 | +0.0827 |
| Orange___Haunglongbing | 0.9941 | 0.9154 | +0.0787 |
| Strawberry___Leaf_scorch | 0.9977 | 0.9406 | +0.0571 |
| Tomato___Bacterial_spot | 0.9564 | 0.9004 | +0.0560 |
| Pepper,_bell___healthy | 0.9826 | 0.9277 | +0.0549 |
| Potato___healthy | 0.9879 | 0.9344 | +0.0535 |

A shared characteristic among these seven classes is that the discriminative signal is *spatially concentrated*: Late Blight presents one or two dominant necrotic patches; Black Rot forms discrete circular lesions; Haunglongbing produces blotchy yellowing on individual leaflets. MobileNetV2's inverted residual bottleneck architecture, which applies depthwise separable convolutions with an expand-project structure, appears particularly effective at encoding these spatially compact features within a narrow channel budget. ResNet-50's standard 3×3 convolutions and identity shortcuts, by contrast, may distribute the same signal across more channels without achieving superior localisation, leading to larger inter-class confusion margins.

### 6.2 Classes Where ResNet-50 Outperforms MobileNetV2

ResNet-50 clearly outperforms MobileNetV2 on only three classes:

| Class | MobileNet F1 | ResNet F1 | Δ |
|---|---|---|---|
| Corn_(maize)___Cercospora_leaf_spot | 0.8440 | 0.9305 | **−0.0865** |
| Corn_(maize)___Northern_Leaf_Blight | 0.8935 | 0.9450 | −0.0515 |
| Tomato___Spider_mites | 0.9229 | 0.9456 | −0.0227 |

These three classes share a common visual characteristic: the diagnostic signal is distributed across a **large spatial region** rather than concentrated in discrete lesion patches. Cercospora and NLB produce elongated streaks that run the full length of corn leaves; spider mite infestation manifests as a fine, stippled discolouration spread across the entire leaf surface. ResNet-50's deeper feature hierarchy — with four residual stages versus MobileNetV2's single feature sequence — may be better suited to integrating long-range spatial context, enabling more reliable recognition of these distributed patterns. This interpretation is consistent with the Grad-CAM comparison in Section 4.2 of the Grad-CAM analysis, which shows that MobileNetV2's attention on Cercospora images, while lesion-focused, fails to capture the specific elongation pattern that distinguishes Cercospora from NLB.

### 6.3 Near-Tie Classes

Fourteen classes show an F1 difference of less than 0.020, indicating that for these classes, architectural choice is effectively immaterial:

| Class | MobileNet F1 | ResNet F1 | Δ |
|---|---|---|---|
| Apple___Cedar_apple_rust | 0.9756 | 0.9769 | −0.0013 |
| Corn_(maize)___healthy | 0.9936 | 0.9979 | −0.0043 |
| Grape___Leaf_blight | 1.0000 | 0.9871 | +0.0129 |
| Tomato___Tomato_mosaic_virus | 0.9878 | 0.9759 | +0.0119 |
| Strawberry___healthy | 0.9934 | 0.9812 | +0.0122 |
| *(and 9 additional close classes)* | | | |

These near-tie classes are predominantly *healthy* variants and well-separated disease classes — categories where visual distinctiveness is high enough that both architectures converge on equivalent representations.

## 7. Tier Analysis

### 7.1 MobileNetV2 Tier Distribution

**Top Tier — F1 ≥ 0.950 (31 classes, 81.6% of taxonomy):**
The large majority of the 38 classes fall into the top tier. This group includes all Apple, Blueberry, Cherry, most Grape, Orange, Peach, Pepper, Potato, Raspberry, Soybean, Squash, Strawberry, and most Tomato classes. The common thread is that these classes either present visually distinctive lesion patterns (e.g., cherry powdery mildew: white powdery surface coating; Orange Haunglongbing: asymmetric chlorotic mottling) or represent healthy leaves with species-specific morphology that is easy to separate from disease classes.

**Middle Tier — F1 0.850–0.949 (6 classes, 15.8%):**
The six middle-tier classes are `Corn_(maize)___Northern_Leaf_Blight` (0.894), `Tomato___Septoria_leaf_spot` (0.940), `Tomato___Spider_mites` (0.923), `Tomato___Late_blight` (0.917), `Tomato___Early_blight` (0.909), and `Tomato___Target_Spot` (0.903). Five of the six are Tomato diseases, underscoring the structural difficulty of the intra-genus discrimination problem. The lone non-Tomato entry (Corn NLB) benefits from Cercospora's false negatives — its recall is 0.994 partly because many Cercospora images are misclassified as NLB.

**Bottom Tier — F1 < 0.850 (1 class, 2.6%):**
`Corn_(maize)___Cercospora_leaf_spot` at F1 = 0.844 is the sole bottom-tier class for MobileNetV2. Its recall of 0.739 is the lowest in the entire classification report, meaning that nearly one in four Cercospora specimens escapes detection. This represents the strongest argument for targeted intervention — either dedicated augmentation of Cercospora training images with diverse lesion lengths and developmental stages, or a hierarchical classification stage that first identifies the disease type (elongated-streak vs. spot vs. blight) before discriminating between specific diseases.

### 7.2 ResNet-50 Tier Distribution

**Top Tier — F1 ≥ 0.950 (20 classes, 52.6%):**
Only 20 of 38 classes clear the 0.95 threshold for ResNet-50, compared to 31 for MobileNetV2. The top tier is largely confined to healthy classes and the most visually distinctive disease categories. The 11-class difference between the architectures' top tiers is the clearest quantitative indicator of MobileNetV2's generalisation advantage.

**Middle Tier — F1 0.850–0.949 (17 classes, 44.7%):**
ResNet-50's middle tier is disproportionately large, containing classes that MobileNetV2 has graduated to the top tier. This tier includes `Apple___healthy` (0.939), `Blueberry___healthy` (0.940), `Corn_(maize)___Cercospora` (0.931), `Grape___Black_rot` (0.906), `Grape___Esca` (0.946), `Orange___Haunglongbing` (0.915), `Potato___Late_blight` (0.925), `Potato___healthy` (0.934), `Strawberry___Leaf_scorch` (0.941), and all six Tomato difficulty classes.

**Bottom Tier — F1 < 0.850 (1 class, 2.6%):**
`Tomato___Late_blight` at F1 = 0.796 is ResNet-50's sole bottom-tier class. A recall of 0.676 in a deployment context would mean that one third of Late Blight infections go undetected — a clinically significant failure rate for a disease that, untreated, can destroy an entire crop within days under humid conditions.

### 8.2 Recommended Report Structure

- **Section 4.1 (Overall Results):** Report the accuracy and macro F1 from the aggregate metrics table above. Cite Fig. D4 for the visual summary. Note that MobileNetV2 achieves higher performance with 10× fewer parameters — this is the core efficiency argument.

- **Section 4.2 (Per-Class Analysis):** Reference Fig. D1 when discussing the distribution of per-class performance. Emphasise that 31/38 classes exceed F1 = 0.95 under MobileNetV2 (top tier), compared to 20/38 under ResNet-50. This provides the statistical backbone for the architecture selection recommendation.

- **Section 4.3 (Failure Mode Analysis):** Use the data from Sections 4 and 5 above. The Tomato-cluster and Corn-elongated-lesion clusters are the two identifiable failure patterns. Connect Corn Cercospora (Section 4.1) to the Grad-CAM confusion analysis (Fig. B1 in the Grad-CAM document) to show both quantitative scale (107 missed out of 410) and qualitative cause (elongated streak ambiguity). Connect Tomato Late Blight to Fig. C1–C2 in the Grad-CAM document to show why MobileNetV2 succeeds where ResNet-50 fails on the same image.

- **Section 5.1 (Architecture Selection Rationale):** Cite Section 6 of this document. The key argument is bidirectional: MobileNetV2 wins on 26 classes while ResNet-50 wins on only 3 (Corn Cercospora, Corn NLB, Spider Mites). Moreover, the 3 ResNet advantages are for distributed-pattern classes where neither architecture is fully adequate (all three are in MobileNetV2's middle tier or at its floor), while the MobileNetV2 advantages include the most clinically significant class in the dataset (Tomato Late Blight, +12 pp).

- **Section 5.2 (Limitations):** Acknowledge that the six Tomato middle-tier classes (Section 7.1) represent a structural limitation of the current approach. A single-stage 38-class classifier cannot resolve intra-genus ambiguities that require host-plant context or sub-lesion texture discrimination. A two-stage pipeline — coarse host identification followed by fine disease classification — is the most principled architectural response to this limitation and is worth noting as a direction for future work.

- **Section 5.3 (Cross-Dataset Robustness):** The per-class analysis reported here applies exclusively to the PlantVillage validation set (controlled laboratory conditions). The PlantDoc evaluation (Phase 3.10) demonstrates that this per-class performance does not transfer to field-condition imagery: overall accuracy drops from 97.13% to 16.02% for the original model, and the Tomato failure classes are among the most severely affected under domain shift.
